# BibleHub AI — Brand Spec

## Design Philosophy
Premium, modern, elegant Bible application. Calm, trustworthy, focused on Scripture. Inspired by Apple, Notion, Arc Browser, Linear, BibleProject, and Scandinavian minimalism.

## Color Tokens

### Light Mode
- `--bg`: #FAFAF8 (Soft Off White)
- `--surface`: #FFFFFF (Pure White)
- `--fg`: #111827 (Near Black)
- `--muted`: #64748B (Slate Grey)
- `--border`: #E2E8F0 (Light Slate)
- `--accent`: #D4AF37 (Warm Gold)
- `--primary`: #16213E (Deep Navy)
- `--secondary`: #2563EB (Royal Blue)
- `--success`: #16A34A (Natural Green)
- `--danger`: #DC2626 (Muted Red)
- `--red-letter`: #B91C1C (Jesus' words)

### Dark Mode
- `--bg`: #1A1D28 (Rich Charcoal)
- `--surface`: #242736 (Dark Surface)
- `--fg`: #F1F5F9 (Off White)
- `--muted`: #94A3B8 (Slate)
- `--border`: #334155 (Dark Border)
- `--accent`: #D4AF37 (Warm Gold)
- `--primary`: #1E3A5F (Deep Navy)
- `--secondary`: #3B82F6 (Blue)
- `--success`: #22C55E (Green)
- `--danger`: #EF4444 (Red)
- `--red-letter`: #FCA5A5 (Light Red)

## Typography
- Display: system-ui, -apple-system, sans-serif (clean Apple/Notion-inspired)
- Body: system-ui, -apple-system, sans-serif
- Mono: ui-monospace, SFMono-Regular, Menlo, monospace

### Scale
- Display/H1: 28–32px font-bold tracking-tight
- H2: 20–24px font-semibold
- H3: 16–18px font-semibold
- Body: 15–16px
- Small: 13–14px
- Caption: 12px

## Layout
- Dashboard: sidebar desktop, bottom nav mobile
- Cards: 12–16px rounded, soft shadow
- Spacing: 8px grid system
- Max content width: 720px (reading), 1024px (dashboard)

## Animation
- Page transitions: 200ms ease-out
- Hover states: 150ms ease
- Card elevation: 200ms ease
- Fade-ins: 300ms ease
- Gentle transitions on all interactive elements

## Icon Style
- Monoline SVG, 1.5–2px stroke
- currentColor
- 20×20px or 24×24px
